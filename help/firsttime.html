First Time Users:
    When the game is run for the first time, the program will bring up an "Open" dialog box asking you for the location of the solitaire bitmap card images.  This is in the subdirectory "bitmap" of wherever you unpacked the solitaire binaries.  After you do this, the game creates a registry entry in HKEY_LOCAL_MACHINE/SOFTWARE/ABI/Solitaire called bitmaps location.  If the software is ever moved, this key needs to be deleted or updated with the new path information.  The easiest way to accomplish this is to do the following:

1  - go to Start > Run
2  - enter "regedit" and press [enter]
3  - A window will appear, on the left there will appear an icon with "My Computer" written next to it.  If, to the left of that icon, there is a "-" sign, skip to step 5.
4  - Click on the + sign next to the "My Computer" icon.
5  - Several folders will appear with capitol names and "+" marks.  Click on the "+" next to "HKEY_LOCAL_MACHINE"
6  - The tree will expand.  Look for "SOFTWARE" and click on the "+" mark. 
7  - The tree will expand again.  Look for "ABI" and click on the FOLDER.
8  - Hit the <Delete> button.  A dialog box will appear asking you to confirm that you want to delete that key.  Click "Yes".
9  - Close the program.

This will delete the key and cause the program to again ask you where the bitmaps are located.  Select the new location. 

NOTE:  The images must be in a subfolder of the game labeled bitmaps in order for the game to work properly.  If, after you have started a new game, you don't see any cards, it is because the location in the registry is incorrect.
