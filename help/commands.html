Command Line Options:
Program usage:   solitaire.exe [options] (filename)

Options:
    /g=gametype where gametype is StreetsAlleys, War, Klondike2
          Starts game with a new game of type gametype

    /h
       Starts game and opens the help window

    /f=filename
       Loads file filename

Filename:
    Loads file (filename).  Same functionality as /f=filename. 

Order of preference:
    If both the /f=filename and a filename is given.  The program will ignore the last filename and attempt to load the file given with the /f option.  The program will attempt to determine the game type from the file.  If no file name is given as either the (filename) option or the /f=filename, the program will look for /g=gametype and start a game of that type (if valid).  [Note: the program only looks at the first letter after the equal sign for the /g option; case is ignored].  If no options are given the game opens but does not start a game type.  For more information on files, see the section labeled Opening / Saving Files.  For more information on the file format see the developer notes section.
