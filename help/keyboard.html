Keyboard Keys:
In general, the following keys are supported.  (Note, key functionality may change during individual game play.)
   
Key
 Function
F1
 Help (This Window)
[CTRL] + Z or Backspace
 Undo Last Move
[CTRL] + Y
 Restart Game
[CTRL] + N or N
 Shuffel (Start a new game)
Up Arrow
 Move pointer up one card or set of cards
Down Arrow
 Move pointer down one card or set of cards
Left Arrow
 Move pointer to the left one card or set of cards
Right Arrow
 Move pointer to the right one card or set of cards
Enter
 Select, move, or play the card that is under the pointer
[CTRL] + O
 Open a file
[CTRL] + S
 Save current game to a file
[CTRL] + X or [ALT] + F4
 Quit
[ALT]
 Activate the menu

The mouse is also supported in this game:
    Left Click - Select / Move / Play card under pointer
    Right Click and Hold - Show card under pointer
    Double Left Click -  Attempt to move the card under the pointer to the game goal.  (Note, this is not supported in all games.  If it is not, it acts like a single left click)
